# Dev Toolkit

一个为开发者打造的高效工具箱，包含针对 Cursor Ultra 套餐的精准额度管理工具 **Cursor Flow**。

## 包含工具

### ⚡ Cursor Flow
针对 Cursor Ultra 用户设计的额度监控面板。

- **工作日算法**：可选择仅计算工作日，让预算分配更科学。
- **智能预测**：基于当前进度计算燃尽率（Burn Rate），提供🔥🔥或❄️❄️的状态反馈。
- **数据持久化**：所有数据存储在本地 LocalStorage，不涉及后端交互。

## 部署

本项目为纯静态站点，可以直接部署到 GitHub Pages。

### 本地预览
```bash
npx http-server .
```
访问 `http://localhost:8080` 即可。
